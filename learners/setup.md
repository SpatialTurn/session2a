## What You Need for This Session

Before starting Session 2a, make sure you have the following ready. If you completed the main workshop setup page, you are already set.

### Accounts

- **USGS EarthExplorer** — for downloading CORONA imagery and SRTM elevation data → [Register here](https://ers.cr.usgs.gov/register)
- **Copernicus Data Space** — for downloading Sentinel-2 satellite imagery → [Register here](https://dataspace.copernicus.eu)

Both accounts are free. Email verification can take a few minutes, so please register ahead of time.

### Software

- **QGIS Desktop (LTR)** — download and install from [https://qgis.org/download/](https://qgis.org/download/). Open QGIS once to confirm it launches.

### Recommended

- **7-Zip** (Windows) or a similar archive tool — needed to extract CORONA `.tgz` downloads. → [Download here](https://www.7-zip.org/)
- A dedicated **project folder** on your computer (e.g., `Desktop/GIS_Workshop/`) to keep downloaded data organized.

### Checklist

- [ ] USGS EarthExplorer account created and verified
- [ ] Copernicus Data Space account created and verified
- [ ] QGIS installed and opens successfully
- [ ] Project folder created
